# 프론트엔드 로컬 서버 실행 방법

## 방법 1: PowerShell 스크립트 실행 (권장)

1. PowerShell을 **관리자 권한**으로 실행합니다.
2. 프로젝트 폴더로 이동:
   ```powershell
   cd c:\dys_prototype
   ```
3. 서버 스크립트 실행:
   ```powershell
   .\start-server.ps1
   ```
4. 브라우저에서 `http://localhost:8000` 접속

## 방법 2: Python이 설치되어 있다면

```powershell
cd c:\dys_prototype
python -m http.server 8000
```

그 후 브라우저에서 `http://localhost:8000` 접속

## 방법 3: Node.js가 설치되어 있다면

```powershell
cd c:\dys_prototype
npx serve .
```

또는

```powershell
npx http-server -p 8000
```

## 방법 4: VS Code Live Server 확장 사용

VS Code를 사용 중이라면:
1. VS Code에서 `index.html` 파일을 엽니다
2. 우클릭 → "Open with Live Server" 선택

## 참고사항

- 서버를 중지하려면 터미널에서 `Ctrl+C`를 누르세요
- 포트 8000이 이미 사용 중이라면 다른 포트(예: 8001, 8080)를 사용하세요
